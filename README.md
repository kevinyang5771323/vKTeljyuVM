# 招聘大数据可视化分析系统 python374

## 项目概述

本项目是基于Python开发的招聘大数据可视化分析系统，主要用于对招聘信息进行数据挖掘、展示与分析，为用户提供岗位、薪资、企业等多维度的数据可视化服务。

## 技术栈

- 前端：暂无具体信息
- 后端：Flask
- 数据库：MySQL

## 功能模块

### 首页数据展示

- 岗位信息数量
- 行业平均薪资
- 行业最低薪资
- 行业最高薪资
- 学历与薪资统计
- 整体学历要求
- 福利待遇
- 热门岗位

### 数据概况

- 开发岗位分类
- 非开发岗位分类
- 薪资预测
- 岗位匹配

### 可视化

- **薪资情况**：通过学历筛选，展示各岗位薪资分布和比例，以柱状图和饼图形式呈现。
- **企业情况**：选择职位，查看主要招聘城市、公司规模和职位比例饼图。
- **学历情况**：展示各职位对学历和工作经验的要求，使用漏斗图和矩形树图进行可视化。

### 智能化

- **薪资预测**：用户可选择职位、学历、公司规模、城市和工作经验，以及不同的预测模型，预测最低薪资。
- **职位匹配**：通过筛选职位要求，为用户匹配最合适的职位信息。
